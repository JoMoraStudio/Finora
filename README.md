# Finora – Persönlicher Finanzplaner

## Funktionen

- Einnahmen, Ausgaben & Sparbeträge tracken
- Budget pro Kategorie festlegen
- Sparkonten mit Zielen verwalten (Aktien, Tagesgeld, Bauspar…)
- Auswertungen: Diagramme, Jahresübersicht, Sparquoten
- Monatsverlauf & Jahresdurchschnitte
- Export & Import als JSON
- Als App auf iPhone, iPad & Android installierbar (PWA)

## Als App installieren

**iPhone / iPad (Safari):**
1. Seite in Safari öffnen
2. Teilen → „Zum Home-Bildschirm"

**Android (Chrome):**
Menü → „App installieren"
